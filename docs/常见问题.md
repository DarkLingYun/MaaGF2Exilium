# 常见问题
## 加载资源失败
删掉整个文件，重新下载新的版本软件的压缩包，重新使用

~~重装解决99%的问题~~

## 无法删除文件夹，被占用了
打开任务管理器，搜索 `adb.exe` 结束进程

## 连接模拟器时发生错误
到最上面一行的“连接”界面，点击长得像笔的那个图标，打开 Adb 编辑器界面

到你所用的模拟器安装路径下，找到 `adb.exe` 这个文件，在 Adb 路径这里选择这个文件

类似下图

![](../imgs/常见问题-1.png)

如果还不行，可以试着：

1. 关闭模拟器和 maa
2. 到任务管理器中查找 `adb.exe`，结束进程
3. 再试一次
